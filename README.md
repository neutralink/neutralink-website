# 🌐 NeutraLink Website

This is the **official institutional website** of [NeutraLink](https://neutralinkeco.com), built to present the platform, mission, technology, and opportunities for users, partners, and investors.

---

## 📄 Overview

The website includes:

- 🌍 Company presentation and mission
- ⚙️ How NeutraLink works
- 💡 Product and platform highlights
- 📈 Carbon credits and sustainability education
- 📰 Blog with relevant news and insights
- 📬 Contact and support section
- 🇧🇷🇺🇸 **Bilingual support** (Portuguese 🇧🇷 & English 🇺🇸)

---

## 🚀 Tech Stack

- **Next.js** (or Astro / Hugo – depending on setup)
- **TypeScript**
- **Tailwind CSS** or ShadCN UI
- **i18n** for language translation (e.g., `next-i18next` or `react-i18next`)
- **Markdown** support for blog posts
- SEO tools (Open Graph tags, sitemap, structured data)

## 🧪 Development Setup

1. Clone the repository:
```bash
git clone https://github.com/neutralinkeco/neutralink-website.git
cd neutralink-website
```

2. Install dependencies:
```bash
npm install
```

3. Create a `.env` file:
```bash
cp .env.example .env
```

4. Run the project locally:
```bash
npm run dev
```
